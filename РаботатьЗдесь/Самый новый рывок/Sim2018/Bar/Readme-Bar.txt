Файл Readme для описания ДвижкаСолюшена.
Текущая версия библиотеки: 1.0.0.0

Общее описание:

Это библиотека движка для работы с Солюшенами Тапп.
Это встраиваемая версия Движка, с БД Microsoft Jet4.0 или без БД.
Только один пользователь монопольно использует Движок.
С начальной поддержкой мультиконтейнерности.


Системные требования:

- Windows XP SP2 архитектуры x86
- .NET Framework 2.0
- ...

Отличия от предыдущих версий:

-Нет. Это первая версия.


Реализуемая функциональность:

* Хранение данных в AccessСУБД.
    * Сменные адаптеры для работы с разными типами БД:
    * АПИ для работы с адаптером БД.
    * Поддерживать режим СолюшенБезБД.

* Библиотека для работы в составе приложения:
    * Внутрипроцессная версия, встраиваемая в С# - проект.
    * При этом в одном процессе может быть несколько одновременно открытых контейнеров.

* АПИ АпиСолюшена для работы с ячейками, связями, контейнером:
    * Создание нового Солюшена и новой БД.
    * Открытие и закрытие существующего солюшена.
    * Оптимизация БД Солюшена.
    * Поддерживать все разновидности ячеек и связей.
    * Создание, чтение, изменение, удаление ячеек.
    * Поиск ячеек по идентификатору, названию, описанию, регулярному выражению.
    * Создание, чтение, изменение, удаление связей.
    * Поиск связей по идентификатору, названию, описанию, регулярному выражению.
