1 Дата начала: 7 мая 2016 г.

2 Название: ДвижокТапп

3 Краткое описание: Разработка механизма графовой семантической базы данных для проектов СИМ.

4 Описание: 

4.1 Обстановка и причины создания проекта: 
Результат проекта необходим для продвижения работ по нескольким проектам.

4.2 Конечный результат проекта:  
Концепция механизма графовой семантической базы данных.
Работоспособный механизм, пригодный для хранения и реструктуризации данных.

4.3 Предполагаемая ситуация для использования проекта: 
Использовать как хранилище данных для некоторых из моих проектов.

4.4 Предполагаемая трудоемкость проекта:  1 год.

4.5 Текущее состояние проекта: Инициализация 

4.6 План работы: 
1) создать инфраструктуру для работы над концепцией проекта.

4.7 Замечания: 
Поскольку это первый мой концептуально планируемый проект, надо основательно  проработать всю инфраструктуру проекта.
Чтобы потом копировать эти решения в следующих проектах.




Стандартная структура каталогов проекта: 

* Описание.тхт	 - файл описания проекта 
* сделатьххх.тхт - файл быстрого описания задачи проекта 
* ПапкаСделатьххх - папка задачи проекта 
  * файлы сырье для переработки.ххх - файлы для задачи 
* Архив - папка бекапов для всех частей проекта 
  * СтарыеВерсииПроекта папка старых версий проекта (если они есть) 
* ХХХWiki - папка вики проекта 
* Материалы - папка для материалов проекта, образцов программ и полезных сведений для переработки 
  * ярлык-ссылка на данные из КоллекцияДанных. (если в проекте есть массивы данных) 
* СправочныеСведения - папка для справочных документов 
* ЗаметкиНаброски - папка для накопления одиночных заметок, которые надо перенести в вики 
* ПапкаМодуляХ	- папка модуля проекта 
* ПапкаМодуляУ	- папка модуля проекта 
  * временные файлы работ проекта 
  * основные файлы работ проекта 
  * файлы диаграмм 
* Утилиты	- папка вспомогательных утилит проекта и их проектов
